
RESPONSIVE SHARE BUTTONS MODULE
-----------------

Maintainer: Tim Wooten (twooten)
Original HTML/CSS/JS code from Saran Chamling
(https://plus.google.com/u/0/114294210195147580398?rel=author).
Code found here: http://www.saaraan.com/2013/09/responsive-share-buttons-css-and-jquery

This module makes it easy to add share buttons to your site,
using the Superfish jQuery plugin

Installation
------------
1. Copy responsive_share_buttons folder to your sites/all/modules directory.
2. At Administer -> Site building -> Modules (admin/build/modules)
enable the module.

3. Configure the module settings at
Administer -> Site configuration -> Responsive Share Buttons (admin/settings/responsive_share_buttons).
